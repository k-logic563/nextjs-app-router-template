Next.js template with App Router.

## Features
- [x] TypeScript
- [x] ESLint(using biome)
- [x] Prettier(using biome)
- [x] Jest
- [x] React Testing Library

## Getting Started
```bash
# install dependencies
npm ci
# start development server
npm run dev
# build for production
npm run build
```

## Not Included
- UI Framework
- CSS Framework
- State Management
